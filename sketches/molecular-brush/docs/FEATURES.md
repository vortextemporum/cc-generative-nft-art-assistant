# Features & Rarity Guide

## Feature Overview

Each artwork's features are deterministically derived from a 64-character hex hash.

## Rarity Tiers

| Tier | Probability | Occurrence |
|------|-------------|------------|
| Common | ~50-60% | Standard outputs |
| Uncommon | ~25-30% | Interesting variations |
| Rare | ~12-15% | Unusual combinations |
| Legendary | ~3-5% | Striking/unique |

## Physics Mode

| Mode | Rarity | Behavior |
|------|--------|----------|
| molecular | common | Lennard-Jones potential forces |
| brownian | common | Random walk with weak forces |
| flocking | uncommon | Boids algorithm (separation, alignment, cohesion) |
| waves | uncommon | Sinusoidal motion patterns |
| orbital | rare | Gravitational orbit around center |
| magnetic | rare | Color-based attraction/repulsion |
| vortex | legendary | Spiral toward center |
| explosion | legendary | Outward burst from center |

## Rarity Combinations
