# BLOB-COOKIE - AI Assistant Guide

## Overview

Exact 2D bezier blob extruded into 3D with beveled edges.
Preserves negative spaces and concave areas.

## Algorithm

1. `buildBlobPoints()` - Exact port of 2D blob (ring + bezier control points)
2. `createBlobShape()` - Convert to THREE.Shape using bezierCurveTo
3. `THREE.ExtrudeGeometry` - Extrude with bevel for smooth edges

## Key Difference from Lathe

- **Lathe** (blob-forms): Revolves profile, fills in concave areas
- **Extrude** (this): Preserves exact shape including negative spaces

## Parameters

### Blob (from original)
- numPoints, baseRadius, radiusRandomness, cpOffsetAngle, cpDistance

### Extrusion
- depth: Cookie thickness
- bevelSize: Edge roundness
- bevelSegments: Bevel smoothness
