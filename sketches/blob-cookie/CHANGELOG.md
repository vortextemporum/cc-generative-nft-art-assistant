# Changelog

## [1.0.0] - 2026-02-03

### Added
- Initial release
- Exact 2D bezier blob algorithm (buildBlobPoints)
- THREE.Shape creation with bezierCurveTo
- THREE.ExtrudeGeometry with bevel
- Preserves negative spaces / concave areas
- Blob params: numPoints, baseRadius, radiusRandomness, cpOffsetAngle, cpDistance
- Extrusion params: depth, bevelSize, bevelSegments
- 7 materials, 4 distortions
- Dev mode UI
