# BLOB-COOKIE

Extruded 2D bezier blobs with smooth beveled edges.

## Why "Cookie"?

Like a cookie cutter - preserves the exact 2D shape including negative spaces and concave areas, just gives it thickness.

## Controls

| Key | Action |
|-----|--------|
| R | Regenerate |
| S | Save PNG |
| Space | Pause |
| L/D | Like/Dislike |

## Parameters

### Blob Shape (original 2D)
- numPoints, baseRadius, radiusRandomness, cpOffsetAngle, cpDistance

### Extrusion
- depth: Thickness
- bevelSize: Edge radius
- bevelSegments: Edge smoothness
