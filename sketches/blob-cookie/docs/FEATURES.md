# Features

## Blob Parameters

| Param | Range | Effect |
|-------|-------|--------|
| numPoints | 4-20 | Complexity |
| baseRadius | 0.5-2.0 | Size |
| radiusRandomness | 0-0.5 | Wobbliness |
| cpOffsetAngle | 0-90 | Curve wobble |
| cpDistance | 0.1-1.5 | Stretch |

## Extrusion Parameters

| Param | Range | Effect |
|-------|-------|--------|
| depth | 0.1-1.5 | Thickness |
| bevelSize | 0-0.3 | Edge radius |
| bevelSegments | 1-12 | Edge smoothness |

## Materials

Matte, Glossy, Chrome, Iridescent, Glass, Emissive, Holographic
