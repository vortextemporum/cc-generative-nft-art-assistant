# Changelog

## [1.0.0] - 2026-02-03

### Added
- Initial release
- EXACT port of 2D bezier blob algorithm (buildBlobPoints)
- Bezier curve sampling for smooth profiles
- THREE.LatheGeometry for 3D revolution
- Original parameters: numPoints, baseRadius, radiusRandomness, cpOffsetAngle, cpDistance
- 7 material types
- 5 distortion types (None, Twist, Taper, Bend, Wave)
- Dev mode UI with sliders for all blob parameters
- Feedback system
