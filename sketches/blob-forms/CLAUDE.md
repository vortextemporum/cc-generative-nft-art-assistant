# BLOB-FORMS - AI Assistant Guide

## Project Overview

**Blob-forms** creates 3D blob objects using the EXACT algorithm from a 2D bezier blob generator, extended to 3D via lathe/revolution.

**Core Algorithm** (ported exactly from original p5.js):
1. Generate N points around a ring at equal angles
2. Each point has randomized radius from center
3. Each point has two bezier control points perpendicular to radial direction
4. Control points have random angular wobble
5. Sample the bezier curve for smooth profile
6. Use THREE.LatheGeometry to revolve around Y axis

**Framework**: Three.js

## Key Functions

| Function | Purpose |
|----------|---------|
| `buildBlobPoints()` | EXACT port of original 2D blob algorithm |
| `sampleBlobProfile()` | Sample bezier curves for smooth profile |
| `profileToLathePoints()` | Convert 2D profile to lathe format |
| `generateBlob()` | Create 3D mesh via LatheGeometry |

## Parameters (from original 2D)

| Parameter | Original | Effect |
|-----------|----------|--------|
| `numPoints` | Points around ring | Blob complexity |
| `baseRadius` | Ring size | Overall size |
| `radiusRandomness` | Radius variation | Wobbliness |
| `cpOffsetAngle` | Control point wobble | Curve smoothness |
| `cpDistance` | Control point distance | Stretch amount |
