## Element Counts by Mode

### QED Elements
- Photon propagators (wavy golden lines)
- Electron/positron lines (straight with arrows)
- Interaction vertices
- Self-energy loops (photon loops)
- Annihilation/creation events

### QCD Elements
- Quark lines (red, green, blue)
- Gluon propagators (curly springs, 8 colors)
- Three-gluon vertices (RGB colored)
- Color confinement tubes
- Color loops

### String Elements
- Open string worldsheets (swept surfaces)
- Closed string worldsheets (torus shapes)
- String vibration modes (harmonic patterns)
- Brane intersections (crossing planes)
- Calabi-Yau manifolds (rare)

### Vacuum Elements
- Virtual particle pairs (bubble loops)
- Quantum foam (dense tiny loops)
- Uncertainty clouds (fuzzy regions)
- Casimir plates (parallel plates with constrained vacuum)
- Vacuum energy fluctuations
